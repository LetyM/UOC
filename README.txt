-------- Proyecto Bacilo Koch -------------
Este proyecto está orientado a la realización de un análisis de datos con información
genética sobre el Bacilo de Koch.

Para ello, contamos con el programa 'main.py' siendo el único que es necesario
ejecutar. Este programa responde a una serie de cuestiones relacionadas con los
ORF (Open Reading Frame) de los genes del Bacilo Koch utilizando las funciones
creadas en 'utilities.py' y representando los resultados mediante las funciones
creadas en 'visual.py'.

Las gráficas creadas en la ejecución de 'main.py' mediante el módulo 'visual.py'
se guardan en el directorio principal bajo la extensión '.png'.

Para el testeo de las distintas funciones, se ha utilizado el código 'test.py'
que recurre a su vez a los archivos de la carpeta 'testdocs', similares a los 
originales pero más simplificados para conocer de antemano los resultados
y poder comprobar el funcionamientos de las funciones.

Por último, en el archivo 'requirements.txt' están listadas las librerías
necesarias para la correcta ejecución del proyecto.
